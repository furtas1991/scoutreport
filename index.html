Backup — all data
Includes everything: reports, players, shadow teams, folders and club crests. Save the file to iCloud or send it via AirDrop. Make a copy before each app update.
Storage
Data is stored offline in this browser using its large-capacity store (typically hundreds of MB to several GB). Your scouting data is tiny — photos are what take space, and there's plenty. Still, export a backup regularly: it's the only thing that protects you if the device is wiped or runs critically low on space.
Data quality
